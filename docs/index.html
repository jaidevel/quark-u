Offcanvas Component

The Offcanvas component provides sliding panels from the left or right of the viewport, controlled via simple data attributes.

How to Use Offcanvas

Follow these steps to set up an Offcanvas panel using Quark-U attributes:

  • Add this attribute to a trigger button: 
    data-q-offcanvas-target="panel-id"
  • Add this attribute to the panel element: 
    data-q-offcanvas="panel-id"
  • Add this attribute to an element inside the panel to close it: 
    data-q-offcanvas-close

Menu Icons

Quark-U includes a flexible, SCSS-driven hamburger menu icon built with pure HTML and utility classes. It supports open/close animation states using BEM modifiers.

Basic Usage

Use the base class .q-menu-icon with three child elements using .q-menu-icon__bar. 

  
<button onclick="this.querySelector('.q-icon-menu').classList.toggle('q-icon-menu--is-open')"
  class="q-navbar__toggle q-d-flex" aria-controls="menu-test">
  <span class="q-icon-menu" aria-hidden="true"></span>
  <span style="display: none;" class="screen-reader-text">Toggle menu</span>
</button>

Live Example

Custom States

Add .q-menu-icon--is-open to trigger the "X" animation. You can toggle this with JS or a class toggle utility. 

            document.querySelector('.q-menu-icon')
              .classList.toggle('q-menu-icon--is-open');

Variants

Future variants can use modifier classes like .q-menu-icon--alt or --bold for alternate styles.

Fluid Typography and Properties

Quark-U provides two mixins for creating fluid responsive values without media queries, using the CSS clamp() function. These are ideal for modern, scalable designs.

1. fluid-font() — Fluid Font Sizes

This mixin generates a fluid font-size that grows smoothly between defined minimum and maximum viewport widths. 

@include fluid-font(1, 1.5); // 1rem to 1.5rem between 360px and 1200px
            @include fluid-font(1.25, 2, 375, 1440); // Custom viewport range

2. fluid-size() — Fluid Other Properties

This mixin can make any property fluid, like padding, margin, width, or height. 

@include fluid-size(padding, 0.5rem, 1.5rem);
            @include fluid-size(width, 300px, 800px);

SCSS Mixins

These utility mixins are available in Quark-U to simplify responsive design and fluid properties.

respond-up

Apply styles when the viewport width is greater than or equal to a specific breakpoint. 

@include respond-up(sm) {
  // Your styles here
}

respond-down

Apply styles when the viewport width is less than a specific breakpoint (minus 0.02px to prevent overlap). 

@include respond-down(md) {
  // Your styles here
}

respond-between

Apply styles between two breakpoints (inclusive range). 


@include respond-between(sm, lg) {
  // Your styles here
}

fluid-property

Create a fluid property (like font-size or padding) that smoothly adjusts based on viewport width. 

@include fluid-property(font-size, 1rem, 2rem);
